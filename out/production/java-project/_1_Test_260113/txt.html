(메인 포함된 클래스가 오직 하나 있고, 그 외에는 나머지 클래스는 , 변수, 메서드들로 구성이 된다. * **메서드(Method)**: 동작 또는 함수 또는 기능. `main()`은 프로그램의 진입점. * **식별자(Identifier)**: 변수, 메서드, 클래스 등의 이름. ### 5. 기본 문법 정의 #### 자바 기본 구조: ```java public class 클래스이름 { public static void main(String[] args) { // 실행 코드 } } ``` #### 식별자 규칙: * 문자, 숫자, 밑줄(`_`), 달러표시(`$`)만 사용 가능 * 숫자로 시작할 수 없음 * 자바 예약어는 사용 불가 * 대소문자 구분 ### 5-2. 많이 사용하는 옵션 및 설명 * `public`: 접근 제한자. 어디서든 접근 가능하게 함 * `static`: 객체 생성 없이 메서드 사용 가능 * `void`: 반환값이 없음을 명시 * `String[] args`: 커맨드라인 인수 받기 위한 배열 ### 6. 원리, 구조 설명 (비유) * **클래스**: 공장에서 찍어낼 설계도면 비유 예시, 붕어빵, 폰, 집, 설계도면을 참고해서, 같은 제품을 찍어서 만들어 낸다. 개념. * **main() 메서드**: 공장을 시작하는 전원 스위치 * **식별자**: 제품에 붙이는 이름표 클래스명, 변수명, 메서드 명. --- ### 7. 실무에서의 활용 * 프로그램은 하나 이상의 클래스로 구성되며, 각 클래스는 책임이 분리됨 * 좋은 식별자는 코드 가독성과 유지보수에 크게 기여함 * 식별자 명명 규칙은 팀 코딩에서 매우 중요함 (네이밍 컨벤션) 캐멀 기법?(HelloWorldHi), 스네이크 형태?(Hello_World_Hi) 기본 예시코드 public class _2_Test1 { public static void main(String[] args) { int score = 90; String name = "이상용"; System.out.println("이름 : " + name + ", 점수 : " + score ); } } 실습1 1)사용자 이름과 나이를 변수로 선언하고 출력하는 프로그램 작성(자기소개) 2) 두 수를 곱한 결과를 출력하는 프로그램 변수, x = 100, y = 200, public class Ex1 { public static void main(String[] args) { String introduce = "본인 소개, 이상용입니다. 나이는 43세입니다. 부산 진구에 살아요"; int number = 100; System.out.println(introduce + "좋아하는 숫자: " + number); System.out.println("=========================="); System.out.println("2)"); int x = 100; int y = 200; int result = x * y; System.out.println("계산 결과 : " + result); } } 7 자바의 데이터 타입과 키 입력 ### 1. 학습 목표 * 자바의 기본 및 참조 데이터 타입을 이해한다. * 다양한 키보드 입력 방법을 배운다. * Scanner 클래스를 이용한 입력 처리 방법을 익힌다. ### 2. 학습 내용 * 자바의 데이터 타입 분류: 기본형(Primitive)과 참조형(Reference) * 정수, 실수, 문자, 논리형 타입의 크기와 범위 * Scanner 클래스 사용법 및 키 입력 처리 예제 실습 --- ### 3. 역사, 배경, 소개 자바는 정적 타입 언어로, 변수 선언 시 데이터 타입을 명시해야 한다. 이는 컴파일러(문법 검사기)가 타입을 체크할 수 있게 하여 안정성과 오류 예방에 효과적이다.
